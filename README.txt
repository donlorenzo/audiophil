Install instructions from source:
=================================

Dependencies:
-------------
 * Python 2.x with x >= 5
 * PyQt4 which itself depends on qt4 namely 
   it need the qt.core, qt.gui, qt.sql and qt.phonon packages
   (http://www.riverbankcomputing.co.uk/)
 * pyuic4 and pyrcc4 which should come with PyQt4
 * mutagen (http://code.google.com/p/mutagen/)

Instructions:
-------------
 * Linux: make && sudo make install
 * Other systems: your on your own, good luck!
